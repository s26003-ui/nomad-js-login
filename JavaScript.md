## 2. JavaScript 기본 문법

### 변수

변수는 데이터를 저장하는 공간이다.  
`let`은 값이 바뀔 수 있는 변수를 만들 때 사용하고, `const`는 값을 바꾸지 않을 변수를 만들 때 사용한다.

### 조건문

조건문은 어떤 조건이 맞는지에 따라 다른 코드를 실행할 때 사용한다.`if`는 조건이 참일 때 실행하고, `else`는 조건이 거짓일 때 실행한다.

### 반복문

반목문은 같은 코드를 여러 번 실행할 때 사용한다.`for`는 반복 횟수가 정해져 있을 때 많이 사용하고,`while`은 조건이 참인 동안 계속 반복한다.

### 함수

함수는 자주 사용하는 코드를 묶어 두는 기능이다.필요할 때 이름을 불러서 사용할 수 있다.JavaScript에서는 함수 선언 방식과 화살표 함수 방식을 많이 사용한다.

### 자료구조

자료구조는 여러 데이터를 저장하고 저장하는 방법이다.

- 배열`[]`
  배열은 여러 값을 순서대로 저장할 때 사용한다.
- 객체`{}`
  객체는 이름표와 값의 형태로 데이터를 저장할 때 사용한다.
