# CSS

## CSS란?

CSS는 Cascading Style Sheets의 약자로, 웹 페이지의 디자인을 담당하는 스타일 언어이다.
HTML이 웹 페이지의 구조와 내용을 만든다면, CSS는 색상, 크기, 간격, 위치 등 화면에 보이는 스타일을 설정하는 역할을 한다.

예를 들어 HTML이 “여기에 제목과 이미지가 있다”는 구조를 만든다면 CSS는 “제목의 색상, 글자 크기, 이미지 모양” 등을 꾸며주는 역할을 한다.

## CSS 기본 문법

CSS는 선택자와 스타일 선언으로 이루어진다.

선택자 {
속성: 값;
}

**구성 요소**

- **선택자(Selector)** : 스타일을 적용할 HTML 요소

- **속성(Property)** : 변경할 스타일 항목

- 값(Value)\*\* : 속성에 적용할 구체적인 설정

- 세미콜론(;)\*\* : 하나의 스타일 선언이 끝났음을 표시

## 주요 선택자 종류

### 태그 선택자

특정 HTML 태그 전체에 스타일을 적용한다.

h1 {
color: red;
}
클래스 선택자

HTML의 class 속성을 이용하여 스타일을 적용한다.
여러 요소에서 중복 사용이 가능하다.

.box {
width: 100px;
}

### 아이디 선택자

HTML의 id 속성을 기준으로 스타일을 적용한다.
보통 하나의 요소에만 사용된다.

#header {
background-color: #eee;
}

## 선택자 결합과 우선순위

### 그룹 선택자

여러 요소에 동일한 스타일을 적용할 때 사용한다.

h1, h2, h3 {
color: blue;
}

### 자손 선택자

특정 요소 내부에 있는 요소만 선택할 수 있다.

div p {
color: gray;
}

### 우선순위

inline style > id 선택자 > 클래스 선택자 > 태그 선택자

## 글자 스타일 설정

CSS에서는 다양한 속성을 이용해 텍스트 스타일을 설정할 수 있다.

- **font-size** : 글자 크기 설정 (px, em, rem 등)

- **font-family** : 글꼴 종류 지정

- **font-weight** : 글자 굵기 설정

- **color** : 글자 색상 설정

## 색상 표현 방법

CSS에서 색상은 여러 방식으로 표현할 수 있다.

색상 이름 : red, blue 등

HEX 코드 : #ff5733

RGB / RGBA : rgb(255, 0, 0) 또는 rgba(255, 0, 0, 0.5)

RGBA는 마지막 값으로 투명도를 설정할 수 있다.

## 배경 스타일

요소의 배경을 꾸밀 때 사용하는 속성이다.

background-color : 배경 색상 설정

background-image : 배경 이미지 설정

background-size : 배경 이미지 크기 조절

## Box Model

웹 페이지의 모든 요소는 사각형 형태의 박스 구조로 구성된다.
이 구조를 Box Model이라고 한다.

## 구성 요소

**Content**

텍스트나 이미지가 들어가는 실제 콘텐츠 영역이다.
width와 height로 크기를 조절할 수 있다.

**Padding**

콘텐츠와 테두리 사이의 안쪽 여백이다.
패딩이 커지면 요소 전체 크기도 함께 커진다.

**Border**

패딩과 마진 사이에 위치하는 테두리 영역이다.
선의 두께, 스타일, 색상을 지정할 수 있다.

**Margin**

요소 바깥쪽에 생기는 외부 여백이다.
다른 요소와의 간격을 조절할 때 사용된다.
