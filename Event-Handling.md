## 이벤트 처리

이벤트 처리는 사용자의 동작에 따라 JavaScript 코드가 실행되도록 만드는 것이다.

- `addEventListener` : 요소에 이벤트를 등록하는 메서드이다.
- `click` : 요소를 클릭했을 때 발생하는 이벤트이다.
- `input` : 입력창의 값이 바뀔 때 바로 발생하는 이벤트이다.
- `change` : 입력값 변경이 완료되었을 때 발생하는 이벤트이다.
- `event` : 발생한 이벤트에 대한 정보를 담고 있는 객체이다.
- `event.target` : 이벤트가 발생한 실제 요소를 가리킨다.
