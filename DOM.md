## DOM 조작

DOM 조작은 JavaScript를 사용해 HTML 요소를 선택하고, 내용을 바꾸거나 새로운 요소를 추가하는 것이다.

- `document.querySelector` : 조건에 맞는 첫 번째 요소 1개를 선택한다.
- `document.querySelectorAll` : 조건에 맞는 모든 요소를 선택한다.
- `createElement` : 새로운 HTML 요소를 만든다.
- `appendChild` : 만든 요소를 화면에 추가한다.
- `innerHTML` : 요소 안의 HTML 내용을 바꾼다.
- `textContent` : 요소 안의 글자 내용을 바꾼다.
