# HTML

## HTML이란?

HTML은 HyperText Markup Language의 약자로, 웹 페이지의 구조를 만들기 위해 사용하는 마크업 언어이다.
웹사이트에서 보이는 글, 이미지, 버튼 등의 요소를 배치하고 페이지의 기본적인 틀을 만드는 역할을 한다.

## HTML 문서의 기본 구조

**DOCTYPE**

<!DOCTYPE html>

이 선언은 브라우저에게 현재 문서가 HTML5 규격을 따른다는 것을 알려주는 역할을 한다.

**html 태그**

<html>
</html>

모든 HTML 요소를 포함하는 가장 바깥쪽 태그이다.

**head 태그**

<head>
</head>

웹 페이지의 제목, 문자 인코딩, 스타일 정보 등 문서에 대한 정보를 담는 영역이다.
이 부분은 브라우저 화면에 직접 표시되지는 않는다.

**body 태그**

<body>
</body>

사용자가 실제로 보게 되는 텍스트, 이미지, 버튼 등의 콘텐츠가 들어가는 영역이다.

## 자주 사용하는 HTML 태그

**div**

div는 페이지 레이아웃을 구성할 때 많이 사용하는 태그이다.
블록 요소로 동작하며 화면에서 큰 영역을 구분할 때 사용된다.

**span**

span은 텍스트의 일부에 스타일을 적용할 때 사용하는 인라인 요소이다.
줄 바꿈이 발생하지 않는 특징이 있다.

**p**

문단을 만들 때 사용하는 태그이다.
텍스트를 하나의 문단 단위로 나눌 때 사용한다.

**a**

다른 페이지나 웹사이트로 이동할 수 있는 하이퍼링크를 만드는 태그이다.
href 속성을 사용하여 이동할 주소를 지정한다.

**img**

이미지를 웹 페이지에 표시할 때 사용하는 태그이다.
src 속성으로 이미지 경로를 지정하고 alt 속성에는 이미지 설명을 작성한다.

**form**

사용자가 입력한 데이터를 서버로 전달할 때 사용하는 태그이다.
로그인이나 회원가입 화면에서 자주 사용된다.

**input**

사용자가 정보를 입력할 수 있는 입력 필드이다.
type 속성에 따라 텍스트 입력, 체크박스, 비밀번호 입력 등 다양한 형태로 사용할 수 있다.

**button**

사용자가 클릭할 수 있는 버튼을 만드는 태그이다.
주로 form과 함께 사용하거나 자바스크립트 기능을 실행할 때 활용된다.

## 시맨틱 태그(Semantic Tag)

시맨틱 태그는 태그의 이름만 보고도 해당 영역의 의미를 알 수 있도록 만든 HTML 태그이다.

**header**

웹 페이지의 상단 영역을 나타낸다.
보통 로고, 메뉴, 검색 기능 등이 포함된다.

**main**

페이지에서 가장 중요한 핵심 콘텐츠가 위치하는 영역이다.
한 페이지에서 한 번만 사용하는 것이 일반적이다.

**section**

비슷한 주제의 콘텐츠를 하나의 영역으로 묶을 때 사용한다.

**footer**

페이지 하단에 위치하는 영역이다.
보통 저작권 정보, 연락처, 사이트 정보 등이 포함된다.
